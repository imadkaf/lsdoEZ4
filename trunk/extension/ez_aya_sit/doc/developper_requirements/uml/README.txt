INTRO: 
Paste in this folder the following uml diagram :
- Use case diagrams : These are used to describe normal usage of the extension /
solution. Use case diagrams should cover all operations documented in the enduser
documentation.
-Activity diagrams : These are a more detailed description of what is happening in
each use case. These are only required for functionality that has end-user
interaction.
- CoLlaboration or Communication diagrams : These are required for all functionality that interacts
with either the eZ Publish API or external services.
